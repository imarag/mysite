Utilize this interactive tool if you have a seismic file (one of the formats supported by Obspy read function) and you need to inspect its contents (header and visualized data values) or update the header information (station name, components, etc.). Begin by uploading your seismic data file. The fields will display the seismic parameters found in the uploaded file's header. You can modify these parameters using the Update Header option, but note that the values for the sampling rate (fs) and total sample points (npts) cannot be changed. If you want to create a seismic MiniSEED file from scratch, use the ASCII To MSEED converter tool. Feel free to download the seismic file's data in CSV format, with components as the data column headers, using the Download Data option. After updating the header, download the file with the updated header using the Download File option.

Keep in mind the following:

  • If your seismic file contains more than 6 records or traces, it will raise an error due to decreased performance.
  • The station name should consist of 2 to 8 characters or numbers.
  • The "components" field should include record components separated by commas. Each component should consist of 1 to 4 characters or numbers. Examples include E, N, Z or X, Y, Z or X1, X2, V or E, N or E or Z. If the "components" field is left empty, default components will be used.
