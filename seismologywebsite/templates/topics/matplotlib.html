Introduction

Matplotlib is a Python library for data plotting. It contains a variety of functions to depict your data utilizing different graph types (e.g., line graph, bar graph, pie chart). In this guide we will present the basic functionality of the library is order for someone to be able to quickly generate a figure without having to spend time searching the vast amount of information presented in the documentation. However, bear in mind that an in-depth knowledge of the library requires reading and understanding the documentation in depth.

Start by importing the matplotlib library:

import matplotlib.pyplot as plt

Basic Matplotlib structure

The basic structure of the Matplotlib library are the matplotlib.pyplot.figure and the matplotlib.pyplot.Axes classes. The matplotlib.pyplot.figure is in a sense, a container for one or multiple matplotlib.pyplot.Axes instances. To begin plotting your data, you must first create these two essential objects. You can initiate them by using the matplotlib.pyplot.subplots function which creates a plot of multiple matplotlib.pyplot.Axes instances as a grid. These matplotlib.pyplot.Axes instances are called subplots in the grid.

This function returns a tuple with a matplotlib.pyplot.figure instance as its first item and a single matplotlib.pyplot.Axes object, or an array of matplotlib.pyplot.Axes objects as its second item. That depends on the number of subplots in your plot grid. 

# initialize a figure and axes instances
                fig, ax = plt.subplots()

The function can get various parameters to control the final graph. These parameters are described below:

  • nrows, the number of rows (default, 1) your plot grid will contain
  • ncols, the number of column (default, 1) your plot grid will have
  • sharex, control whether the plot subplots will have shared x-axis vertically (True) or not (default, False)
  • sharey, control whether the plot subplots will have shared y-axis horizontally (True) or not (default, False)
  • figsize, the size ([with, height]) of the figure in inches (default, [6.4, 4.8])
