ASCII to MSEED

Convert raw ASCII data to seismic waveform binary data

If you have ASCII files that you would like to use with our tools, we offer a convenient conversion tool that allows you to convert ASCII files to the MSEED file format which ensures compatibility with our interactive tools. This tool supports only ASCII files and more specifically the formats: .csv, .text and .dat files. Start by selecting the appropriate file parameters according to your file format (delimiter, number of rows to read, etc.) and then continue with the second set of parameters that are related to the seismological record (sampling frequency, station name, etc.).
